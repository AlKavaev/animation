animation

Introduction

This is an R package to create and export animations to a variety of formats (HTML/JS, GIF, Video, PDF), and it also serves as a gallery of statistical animations. See some live examples here.

Installation

To install the stable version on CRAN:

install.packages('animation')

To install the GitHub version:

if (!require('devtools')) install.packages('devtools')
install_github('animation', 'yihui')

This package is generated by roxygen2:

install.packages('Rd2roxygen')
library(Rd2roxygen)
rab('animation', install = TRUE)
